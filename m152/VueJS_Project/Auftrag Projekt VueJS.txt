Ziel: Notenverwaltungstool 

Benutzer können pro Fach mehrere Noten erfassen. Das Tool zeigt pro Fach den Notenschnitt sowie 
den gesamten Schnitt an. 

Die Fächer sind vordefiniert, verwenden Sie dafür die Fächer Ihres aktuellen Stundenplanes. Es soll 
jedoch einfach möglich sein ein neues Fach hinzuzufügen. 

Die Noten pro Fach werden auf 2 Nachkommastellen gerundet, die Gesamtnote auf 0.5 
(mathematisch). 

Jede Note kann wieder aus der Liste gelöscht werden. Editieren ist nicht möglich, die Note muss 
dabei neu erstellt werden. 

Die App soll nach den Kriterien der Gestaltung «schön» aussehen und sowohl auf Desktop wie auf 
Mobilgeräten verwendbar sein. 

(Optional) Die Noten sollen im localStorage des Browsers gespeichert werden sodass nach einem 
Refresh die Daten noch vorhanden sind. 

Frameworks: VueJS Pflicht, CSS Framework nach Wahl.
